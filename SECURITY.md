# Security Policy
## Supported versions
Only the latest release on the `master` branch receives security fixes.
## Reporting a vulnerability
Please **do not** open a public GitHub issue for security vulnerabilities.
Instead, contact the maintainers privately via the GitHub
[Security Advisories](https://github.com/NeuropsyOL/VIEWA/security/advisories/new) page
or by e-mailing the repository owner directly.
We aim to respond within **7 business days** and will coordinate a fix and disclosure timeline with you.
